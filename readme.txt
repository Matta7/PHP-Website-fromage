Projet étudiant réalisé par Vallée Mathieu et Willenbucher Gurvan.

Site en PHP utilisant l'architecture MVCR (Model View Controller Router).
Il répertorie quelques fromages, et permet à des utilisateurs connectés d'en créer.

Il utilise une base de donnée MySQL, les bases de données sont fournies avec le fichier 'dump'


Pour lancer ce site, il vous faut modifier dans le fichier 'fromage.php'
le DNS, User et Password de votre base de donnée.

Ce site a été testé en localhost (adresse 127.0.0.1)
